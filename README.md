
# 📚 MAD Lab Experiments Repository

This repository contains all Mobile Application Development (MAD) lab experiments including guides, code implementations, and complete projects.

---

## 📂 Repository Structure

Each experiment includes:
- **Guide file** → Steps and theory for the experiment  
- **Code file** → Ready-to-use implementation  
- **Project folder** (where available) → Complete Flutter/Web project

## 🚀 How to Use

### Option 1: Guide + Code Files Approach

1. Open the experiment guide file  
2. Read and understand the steps  
3. Open the corresponding code file  
4. Copy and paste code into your project  
5. Run and test the project  

---

### Option 2: Use Ready Projects

#### Experiment 2 - UI App
```bash
cd Exp2_ui_app/
flutter run
```

#### Experiment 3-4 - Layout App
```bash
cd Exp3_or_4_layout_app/
flutter run
```

#### Experiment 5 - Form App
```bash
cd Exp5_form_app/
flutter run
```

#### Experiment 7 - JSON App
```bash

```

#### Experiment 9 - E-commerce PWA
```bash

cd calculator\ flutter/

```
