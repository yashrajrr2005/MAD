
# 📚 MAD Lab Experiments Repository

This repository contains all Mobile Application Development (MAD) lab experiments including guides and code implementations.

---

## 📂 Repository Structure

Each experiment includes:
- Guide file → Steps to perform the experiment  
- Code file → Ready-to-use implementation  

Some experiments also include complete projects:
- `calculator flutter/` → Full Flutter calculator app  
- `ecommerce-app/` → E-commerce PWA project  
- `Experiment 7 Include File and Json data in flutter/` → JSON-based Flutter app  

---

## 🚀 How to Use

### Option 1: Use Guide + Code Files

1. Open the experiment guide file  
2. Follow the steps  
3. Open corresponding code file  
4. Copy and paste code into your project  
5. Run the project  

---

### Option 2: Use Ready Projects

#### Calculator (Experiment 11)
- Open folder: `calculator flutter/`  
- Run:
```

flutter run

```

---

#### E-commerce PWA (Experiment 9)
- Open folder: `ecommerce-app/`  
- Open `index.html` in browser  
- Can be deployed on GitHub Pages  

---

#### JSON Flutter App (Experiment 7)
- Open folder: `Experiment 7 Include File and Json data in flutter/`  
- Run:
```

flutter run

```
